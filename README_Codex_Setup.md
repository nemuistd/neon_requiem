# NEON REQUIEM Codex Docs

この zip は、Codex に NEON REQUIEM の MVP 実装を依頼するための初期資料です。

## 配置方法

リポジトリルートに以下のように置いてください。

```text
your-repo/
  AGENTS.md
  docs/
    concept.md
    mvp_spec.md
    codex_tasks.md
```

## 最初に Codex へ渡す指示

まずは `docs/codex_tasks.md` の **Task 0** を Codex に渡してください。

重要なのは、いきなり実装させず、最初は資料を読ませて計画だけ出させることです。
