# 16 序盤バランス確認・調整タスク

## 目的

ネオン掲示板、地下礼拝堂、アイテム追加後の序盤バランスを確認し、手触りを整える。

## 作業内容

### 1. 簡易バランス表を作る

`docs/balance_notes.md` を追加する。

記載する内容:
- 各施設のbaseCost
- costMultiplier
- productionPerLevel
- 解放条件
- 各歌のcost / unlock / effect
- 各アイテムのcost / unlock / effect
- 想定到達順
- 手動ライブのみの場合の到達感
- 自動生産込みの場合の到達感

### 2. 序盤の到達時間を概算する

厳密なシミュレーターでなくてよい。
ただし以下を確認する。

- 路地裏ステージLv1までが遠すぎないか
- 路地裏ステージLv5までが遠すぎないか
- ネオン掲示板解放までが遠すぎないか
- アイテム1個目が自然に購入されるか
- 地下礼拝堂が早すぎないか

### 3. 必要に応じて小調整

調整候補:
- ネオン掲示板baseCost
- ネオン掲示板productionPerLevel
- アイテムcost
- アイテム解放レベル
- 地下礼拝堂baseCost
- 地下礼拝堂解放条件

## 注意

- バランス調整は小さく行う。
- 世界観文言を変更しない。
- UI構造を変更しない。
- このPRでは新施設・新アイテムを追加しない。

## 完了条件

- `docs/balance_notes.md` が追加される。
- 必要なら小さな数値調整が入る。
- `npm run build`
- `npm run check:content`
- テストがある場合は `npm run test`
