# Phase 20: 施設進行戦略docs更新

## 目的

施設多数追加を前提に、施設カテゴリと地下礼拝堂の扱いをdocsで固定する。
このPRでは施設実装を変更しない。

## 作業

- `docs/planning/facility_progression_strategy.md` を更新する。
- 表層復興施設、導線施設、記憶施設、深層施設、転生後施設を整理する。
- 地下礼拝堂は削除しないが、序盤メイン導線に固定しない方針を明記する。
- ネオン掲示板後に、薄明通り案内所や記憶施設を挟む案を検討する。
- 地下礼拝堂を分岐施設または中盤以降の深層施設にする案を推奨として記述する。

## 禁止

- 施設解放条件の変更
- 新施設の実装
- 地下礼拝堂の削除
- 地下礼拝堂を序盤必須施設へ戻すこと

## 検証

```sh
npm run build
npm run check:content
npm run test
```
