# Codexタスク06: アイテムタブMVP

## 目的

READMEの今後予定にあるアイテムタブを、最小実装してください。

## 作業内容

1. `src/content/items.ts` を追加してください。
2. `ItemId`, `ItemDefinition`, `ItemState` を追加してください。
3. `GameState` に `items: Record<ItemId, ItemState>` を追加してください。
4. 初期状態生成と保存正規化にitemsを追加してください。
5. UIに「アイテム」タブを追加してください。
6. アイテム購入処理を追加してください。
7. 最初のアイテムを3つ追加してください。

## 初期アイテム案

### 古いネオン管

- id: `oldNeonTube`
- cost: 150 灯るさ
- unlock: 路地裏ステージ Lv 3
- effect: 施設生産 x1.05 または 路地裏ステージ生産 +10%
- flavor: 古いがまだ光るネオン管。路地裏ステージの輪郭を少しだけ強くする。

### 手書きの告知ポスター

- id: `handwrittenPoster`
- cost: 300 灯るさ
- unlock: ネオン掲示板 Lv 1、または路地裏ステージ Lv 5
- effect: ライブ1回の灯るさ +1
- flavor: 名前を見える場所に残すためのポスター。まだ深層語は使わない。

### 録音済みの短い挨拶

- id: `recordedGreeting`
- cost: 800 灯るさ
- unlock: 仮設配信ブース Lv 1、またはネオン掲示板 Lv 3
- effect: オフライン報酬 x1.05、またはオフライン上限 +1時間
- flavor: 不在の時間にも、誰かがその声を思い出せるようにする短い録音。

## 注意点

- このタスクではアイテム効果は最小限でよい。
- すでにEffect共通化が終わっているなら、それを使う。
- Effect共通化が未完了なら、直接処理でもよいが、後でEffectへ移せる構造にする。
- UIは既存のカード表示に合わせる。

## 完了条件

- アイテムタブが表示される。
- 未解放アイテムに解放条件が表示される。
- 購入可能アイテムを購入できる。
- 購入済み状態が保存される。
- 少なくとも1つのアイテム効果が実際に効く。
- `npm run build` 成功。
- `npm run check:content` 成功。
