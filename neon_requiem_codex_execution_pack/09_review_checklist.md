# レビュー用チェックリスト

各Codexタスク完了後、PRレビュー前に確認してください。

## 共通

- [ ] `npm run build` が成功している。
- [ ] `npm run check:content` が成功している。
- [ ] 変更内容の要約がある。
- [ ] 変更ファイル一覧がある。
- [ ] 未完了事項がある場合、明記されている。
- [ ] 依存が追加されていない。追加された場合は理由がある。
- [ ] TypeScript strict を弱めていない。
- [ ] `window.__NEON_DEBUG__` が維持されている。
- [ ] セーブ削除機能が維持されている。

## 世界観

- [ ] `docs/worldbuilding.md` と矛盾していない。
- [ ] 未記載の重要設定を勝手に確定していない。
- [ ] 序盤UIに「祈念工学」「アンカー」「聖歌」などが出すぎていない。
- [ ] 「聖歌」が通常システム名になっていない。
- [ ] 地下礼拝堂を削除していない。
- [ ] ゲーム体験が搾取・絶望中心ではなく、回復・居場所づくり・復興中心になっている。

## ゲーム挙動

- [ ] 新規セーブで開始できる。
- [ ] ライブボタンで資源が増える。
- [ ] 施設強化で資源が消費される。
- [ ] 自動生産が動く。
- [ ] オフライン報酬が動く。
- [ ] 施設解放条件が表示される。
- [ ] アイドル解放条件が表示される。
- [ ] 歌/楽曲の購入が動く。
- [ ] 記録の解放・既読保存が動く。

## UI

- [ ] タブ切替が動く。
- [ ] 主要ボタンのdisabled状態が正しい。
- [ ] 表示名が「灯るさ」方針に沿っている。
- [ ] メッセージ文言が不自然でない。
- [ ] モバイル表示を大きく壊していない。
