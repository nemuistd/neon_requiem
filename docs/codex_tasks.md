## Task 4: 路地裏ステージ強化を実装する

```text
路地裏ステージの強化を実装してください。

要件:
- 路地裏ステージは Lv 0 から始まる。
- 基本コストは 10 拍手。
- コスト式は floor(10 * 1.15 ^ level)。
- 拍手が足りていれば、強化でコスト分の拍手を消費し、路地裏ステージ Lv を 1 上げる。
- UI に現在 Lv と次の強化コストを表示する。
- 拍手が足りないときは、強化ボタンを無効化するか、強化できないことが明確に分かるようにする。

このタスクでは、必要以上に自動生産や保存を実装しないでください。
```
