# docs/fiction について

このフォルダは、ユーザー確認済みの物語・コンテンツ正史を置く場所である。

実装時の参照順は以下を基本にする。

1. `world_design.md`: 章構造、廻、ミステリーアーク、後半開示。
2. `meguri_after_design.md`: 廻後の物語フェーズ、転生回数、燐の先出しと登場設計。
3. `content_design.md`: 施設、歌、アイテム、アイドル、効果タイプ、解放条件。
4. `idol_profiles.md`: アイドル人物像とアイドルの様子草稿。
5. `records_draft.md`: 追加記録テキスト草稿。
6. `a_place_where_names_remain.md` / `story_notes.md`: Ch.1の散文表現と補完メモ。

`docs/current_spec.md` は現在の実装状態、`docs/worldbuilding.md` は用語体系、`docs/fiction/` は承認済みの物語・コンテンツ正史として扱う。実装済みでない内容は、正史ではあるが未実装として区別する。

`world_design.md` で定義された **廻（めぐり）** は正式システム名である。
