# NEON REQUIEM 手動確認チェックリスト

## 基本

- アプリがローカルで起動する。
- `npm run build` が成功する。
- `npm run check:content` が成功する。
- 新規セーブで灯るさが 0 から始まる。
- Console の `window.__NEON_DEBUG__.resetSave()` で初期状態に戻せる。
- 設定画面からセーブを削除できる。

## 初回3分導線

- 新規セーブで起動後、最初に「灯るさ」と「ライブする」ボタンが視認できる。
- 起動直後のメッセージで、路地裏ステージをもう一度人が来られる場所に戻す雰囲気が伝わる。
- 「ライブする」を1回押すと、灯るさが増え、灯里の歌によって路地に灯るさが戻ったことが分かるメッセージが出る。
- 灯るさ不足の状態で強化や購入を試すと、次にライブで灯るさを戻せばよいことが分かる。
- 灯るさ10到達後、路地裏ステージを強化できる。
- 路地裏ステージ強化後、灯るさが消費され、毎秒生産が0より大きくなる。
- 30秒ほど何も押さなくても、毎秒生産で灯るさが増える。
- 初回3分の範囲で、祈念工学、アンカー、祈念負荷、聖歌を知らなくても目標が分かる。
- 初回3分の範囲で、プレイヤーがアイドルを所有・搾取しているように見える文言がない。

## ライブ / 生産 / 保存

- 「ライブする」を押すと灯るさが 1 増える。
- 灯るさ 10 で路地裏ステージを強化できる。
- 施設強化時に灯るさが消費される。
- 施設強化後、秒間灯るさ量が増える。
- クリックしなくても時間経過で灯るさが増える。
- ページ再読み込み後も灯るさと施設 Lv が残る。
- 離れていた時間に応じてオフライン灯るさが加算される。

- 名前の記録壁 Lv3 到達後、地下礼拝堂が解放される。

## 歌

- 歌タブを開ける。
- 関連施設が未解放の歌は表示されない。
- 関連施設が解放済みで条件未達成の歌は、汎用名「未確認の歌」と解放条件で表示される。
- 路地裏ステージ Lv5 かつ 80 灯るさで「路地裏のイントロ」を取得できる。
- 「路地裏のイントロ」取得後、ライブ1回の獲得灯るさが 2 になる。
- 地下礼拝堂 Lv2 かつ 25000 灯るさで「礼拝堂のハーモニー」を取得できる。
- 地下礼拝堂 Lv8 かつ 90000 灯るさで「薄明のコーラス」を取得できる。
- 地下通路修復区画 Lv3 かつ 80000 灯るさで「修復の仮歌」を取得できる。
- 廻1では、名前のない劇場 Lv1 相当の状態でも「最後の名前」が歌タブに表示されない。
- 廻2以降、名前のない劇場 Lv1 かつ 20000000 灯るさで「最後の名前」を取得できる。
- 生産倍率系の歌を取得すると秒間灯るさ量が増える。
- 「修復の仮歌」取得後、オフライン報酬倍率が増える。
- 「修復の仮歌」取得後、地下通路修復区画 Lv5 と合わせて再固定中枢の解放条件を満たせる。
- 「最後の名前」取得後、施設生産倍率と記憶断片の獲得見込みが増える。
- 購入済み歌がリロード後も購入済みとして残る。

## アイテム

- アイテムタブを開ける。
- 関連施設が未解放のアイテムは表示されない。
- 関連施設が解放済みで条件未達成のアイテムは、汎用名「未確認の備品」と解放条件で表示される。
- 路地裏ステージ Lv3 かつ 100 灯るさで「古いネオン管」を購入できる。
- 路地裏ステージ Lv4 かつ 140 灯るさで「手書きの告知ポスター」を購入できる。
- 「手書きの告知ポスター」購入後、ライブ1回の獲得灯るさがさらに 1 増える。
- 路地裏ステージ Lv5 かつ 220 灯るさで「半券の束」を購入できる。
- 「半券の束」購入後、ライブ1回の獲得灯るさがさらに 1 増える。
- 路地裏ステージ Lv6 かつ 260 灯るさで「携帯スポットライト」を購入できる。
- 「携帯スポットライト」購入後、毎秒灯るさの表示が増える。
- ネオン掲示板 Lv2 かつ 500 灯るさで「交代用連絡ボード」を購入できる。
- 「交代用連絡ボード」購入後、オフライン報酬は基礎50%に `x1.10` が乗り、同じ時間のオンライン自然生産の55%になる。
- ネオン掲示板 Lv3 かつ 600 灯るさで「録音済みの短い挨拶」を購入できる。
- 地下通路修復区画 Lv1 かつ 50000 灯るさで「修復用の工具一式」を購入できる。
- 「修復用の工具一式」購入後、毎秒灯るさの表示が増える。
- 購入済みアイテムがリロード後も購入済みとして残る。

## 記録

- 記録タブを開ける。
- 初期解放の記録が読める。
- 未解放記録のタイトルと解放条件は表示されない。
- 施設Lvや歌取得に応じて記録が解放される。
- 地下礼拝堂 Lv8 かつ「礼拝堂のハーモニー」取得後、「バインダー封書・開封」が解放される。
- 初回廻後、深層観測所 Lv1 で「深層観測所・前室報告」が解放される。
- 初回廻後、深層観測所 Lv2 で「澪・霞の観測記録」が解放される。
- 初回廻後、工学記録保管区 Lv2 で「祈念工学・記録断片」が解放される。
- 初回廻後、祈念工学実験跡地 Lv1 で「実験跡地・現地調査報告」が解放される。
- 廻2以降、名前のない劇場 Lv1 で「名前のない劇場・残響記録」と「燐・最初の言葉」が解放される。
- 廻2以降、名前のない劇場 Lv3 と「最後の名前」取得後、「バインダー封書・全文」が解放される。
- 廻2以降、「最後の名前」取得後、「名前のない劇場・残響公演」が解放される。
- 記録を読むと既読になる。
- 既読状態がリロード後も残る。
- 廻後バフ取得後、対象記録の末尾に追記が表示される。
- 記録追記を読むと追記未読バッジが消え、リロード後も追記既読状態が残る。
- 記録を読んでも灯るさや生産量は変化しない。
- surface の記録に、祈念工学、アンカー、聖歌、祈念負荷が出ていない。

## 廻

- 再固定中枢 Lv1 到達後、廻タブでリセット対象、引き継ぎ対象、記憶断片の獲得見込み、次の記憶断片までのゲージが表示される。
- 廻実行ボタンを押すと確認ダイアログが出る。
- 確認後、灯るさ、施設レベル、歌、アイテム、交流がリセットされる。
- 記録の既読状態、追記既読状態、廻回数、記憶断片、廻後バフは残る。
- 初回廻後、ヘッダーに記憶断片が表示される。
- 廻直後は専用の清算画面に入り、清算中であることが明確に表示される。
- 清算中のみ、記憶断片を使って廻後バフを購入できる。
- 清算中に追記が増えた記録がある場合、清算画面に対象記録が表示され、控えめな「記録タブで追記を読む」導線から清算を閉じて記録タブへ移れる。
- 清算中にリロードしても清算画面へ戻る。
- 「通常進行へ戻る」を押すと清算フェーズが閉じ、通常プレイ中は廻後バフを購入できない。
- 廻で既読記録を引き継いだだけでは記録通知が出ず、追記が表示された記録には追記未読通知が出る。
- 交流20以上で廻を実行したアイドルには、アイドル表示に「以前の廻の痕跡」が出る。
- 廻後にリロードしても、記憶断片、廻回数、購入済みバフ、追記表示が復元される。
- Ch.9 収束条件を満たすと、廻タブに「再び灯るものへ」が表示される。
- 「再び灯るものへ」から「復興を続ける」で復興区画へ戻れる。
- 「再び灯るものへ」から「もう一度廻る」を選ぶと、通常の廻確認ダイアログが表示される。
- モバイル幅でも廻タブ、廻実行パネル、廻後バフ一覧が大きく崩れない。
